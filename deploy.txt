# Arabic Writing Improvement Platform – Full Project Spec

## 1. Project Overview

**Goal:**
A web platform that helps **students improve their Arabic writing skills** through:

1. Reading a **topic/lesson** about a specific writing style or theme.
2. Seeing a **sample/model** of that type of writing.
3. Interacting with:

   * A **predefined question bot** (fixed Q&A about the topic).
   * An **AI chat box** (free conversation with AI about the topic).
4. Writing their own **text** for that topic and type of writing.
5. Receiving **evaluation** from:

   * The **AI** (automatic feedback).
   * The **teacher** (manual review and comments).

There are **two main roles**:

* **Student**
* **Teacher**

The platform is **front-end only for now** (React + Vite + TypeScript), but the structure must be ready for backend and AI integration later.

* **AI Integration:**

## 5. Theme & Layout

### 5.1 RTL + Arabic

* `html` and `body` should use: `direction: rtl;` and `lang="ar"`.
* Global font: use a modern Arabic font (e.g., **Cairo** or similar).
* All UI text labels are in Arabic.

### 5.2 Theme System

Use `useTheme.ts` hook to:

* Store theme in state: `"light"` or `"dark"`.
* Apply class `dark` on `<body>` (or main root) when theme is dark.
* Use CSS variables in `global.css`, e.g.:

```css
:root {
  --bg-main: #F5F6F7;
  --bg-card: #FFFFFF;
  --text-main: #0A2540;
  --text-muted: #5D6B82;
  --primary: #133B5C;
  --accent: #1B6CA8;
}

body.dark {
  --bg-main: #0B1A2A;
  --bg-card: #112A42;
  --text-main: #FFFFFF;
  --text-muted: #A5B4C4;
  --primary: #1B6CA8;
  --accent: #34D399;
}
```

`Navbar` should include a **theme toggle button** to switch between light/dark using `useTheme`.

## 7. Hooks

### 7.1 `useTheme.ts`

Responsibilities:

* Maintain `theme` state (`"light"` or `"dark"`).
* Sync with `localStorage` (optional but preferred).
* Add/remove `dark` class on the `document.documentElement` or `body`.

Example shape:

```ts
type Theme = "light" | "dark";

export function useTheme() {
  const [theme, setTheme] = useState<Theme>("light");
  // logic to toggle and persist
  return { theme, toggleTheme };
}
```
